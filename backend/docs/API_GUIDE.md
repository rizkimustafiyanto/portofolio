# API Guide

Dokumentasi ini dipisah supaya lebih gampang dipakai:

- `API_WORKFLOW.md` untuk langkah praktis membuat API/module baru.
- `API_REFERENCE.md` untuk penjelasan konsep GraphQL, middleware, `DTO`, `model`, `pkg`, dan pola yang dipakai repo ini.

Kalau kamu baru mulai nambah fitur, baca `API_WORKFLOW.md` dulu.
Kalau kamu sedang bingung soal konsep seperti `extend type`, directive `@auth`, atau relasi tabel, buka `API_REFERENCE.md`.
